

## 📌 프로젝트 개요

**보이스쉴드**는 50~70대 사용자가 실제 보이스피싱 상황을 체험하며 대응 능력을 기를 수 있는 **AI 기반 예방 학습 플랫폼**입니다.

사용자는 기관 사칭, 가족 사칭, 택배 문자, 메신저 피싱, 투자 사기 등 다양한 보이스피싱 사례를 카드형 콘텐츠와 대화 시뮬레이션을 통해 학습할 수 있으며, AI가 학습 결과를 분석하여 개인에게 필요한 맞춤형 피드백과 추천 학습을 제공합니다.

또한 PWA 기반으로 제공되어 별도의 앱 설치 없이 스마트폰에서 쉽게 이용할 수 있으며, 실제 보이스피싱이 발생하는 모바일 환경과 유사한 경험을 제공합니다.


## ✨ 주요 기능

#### 회원 관리
- 카카오 로그인 / 게스트 모드
- 기본 프로필 생성 및 학습 기록 저장

#### 사례 학습
- 보이스피싱 사례 목록 및 상세 조회 (카테고리·난이도·예상 학습시간 표시)
- 카테고리별 학습 (기관 사칭, 가족 사칭, 택배 사칭, 메신저 피싱, 투자 사기 등)
- TTS 음성 제공, 실제 문자·통화·이미지 예시 확인

#### 대화 시뮬레이션
- 보이스 모드(TTS 기반 전화 시뮬레이션) / 메시지 모드(텍스트 기반 메신저 피싱) 선택
- 선택형 대응 학습 및 사용자 선택 로그 저장
- AI 맞춤 피드백 제공

#### 퀴즈
- 객관식 퀴즈 풀이 및 정답 저장
- 오답 확인 및 복습, 다시 풀기

#### 학습 기록
- 학습 진행률 및 완료한 시나리오 확인
- 취약 유형 분석, 최근 학습 기록 조회

#### 신고 안내
- 보이스피싱 신고 방법 및 긴급 연락처(112, 1332) 안내
- 실제 대처 방법 및 예방 수칙 확인

#### 설정
- 글자 크기 / TTS 속도·음성 변경
- 학습 알림 시간 설정, 홈 화면 추가(PWA 설치)
- 회원탈퇴


## 🛠 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
